Веб-приложение по примеру из книги Адама Фримена "ASP.NET Core Mvc 2 с примерами на C# для профисcоналов 7-е издание" 
построенное на версии фреймворка 2.2 и с помощью Visual Studio 2019

Основные отличия от примера в книге:
* Вместо метапакета Microsoft.AspNetCore.All используется Microsoft.AspNetCore.App
* Вместо менеджера библиотек клиентской стороны bower используется libman
